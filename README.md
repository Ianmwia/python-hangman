# Hangman Game

Hangman is a game in which a player tries to guess a letter of a word to test spelling and word deduction skills

![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=fff)

## How to play

Run the command in terminal

```.py
python hangman.py

```

You as a player have a number of chances to guess the letters of the word one by one, If you guess all the letters right, you win the game, else you lose and the incorrect word is displayed
